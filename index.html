vue上线部署环境

1、首先我们在写vue项目时，要搭建好基本的环境
(1)、从node官网下载并安装node，安装步骤很简单，只要一路“next”就可以了。
安装完成后，打开命令行工具输入命令node -v，如果出现对应版本号，就说明安装成功了。
(2)、点击进入淘宝的cnpm网站,里面有详细的配置方法
或者直接在命令行输入：
npm install -g cnpm –registry=https://registry.npm.taobao.org
输入cnpm -v，可以查看当前cnpm版本，这个和npm的版本还是不一样的。

2、 在用 vue.js 构建大型应用时推荐使用 npm 安装，npm 能很好地和诸如 webpack 或 browserify 模块打包器配合使用。vue提供了vue脚手架vue-cli，可用于快速搭建大型单页应用。
npm install –global vue-cli
cnpm install –global vue-cli

##创建一个基于 webpack 模板的新项目

3、初始化一个项目 （运行初始化命令的时候可以输入几个基本的选项，如项目名称，描述，作者等信息，如果不想填直接回车默认就好，项目的名称不能大写，不然会报错。）
vue init webpack my-project

4、安装项目所需要的依赖
npm install
cnpm install

5、运行项目 (出现Welcome to Your Vue.js App；如果看到这个界面，说明配置成功)
npm run dev

##接下来就是和后端配合调试数据
6、前端开发过程中使用后台的API的话，可以通过配置proxyTable来将相应的后台请求代理到专用的API服务器；
在config=>index.js=>修改proxyTable如下代码所示:
module.exports = {
dev: {
// Paths
autoOpenBrowser: false,
assetsSubDirectory: ‘static’,
assetsPublicPath: ‘/‘,//上线时将”/“更换成’./‘
proxyTable: {
‘/api’:{
target: ‘http://178.12.3.98:8089',//178.12.3.98所要连接的服务器ip地址
changeOrigin:true,
pathRewrite:{
‘^/api/‘:’’
}
}
},

7、vue组件化
具体操作请参考https://cn.vuejs.org/v2/guide/installation.html

8、vue提供了类似于javascript的ajax，axios基于 Promise 的 HTTP 请求客户端，可同时在浏览器和 node.js 中使用；
npm install axios；具体参考https://www.npmjs.com/package/vue-axios
配置：
在main.js中import axios from ‘axios’；Vue.prototype.$axios = axios
调用：
//get请求
axios.get(‘/user?ID=12345’)
.then(function (response) {
console.log(response);//所请求得到的数据
})
.catch(function (response) {
console.log(response);
});
跨域:
headers:{
‘Access-Control-Allow-Origin’:’*’
}
